JUST RUN THESE 2 COMMANDS IN ORDER:

================================================

COMMAND 1 - Delete node_modules and reinstall:

For Windows PowerShell:
rmdir /s /q node_modules
npm install --legacy-peer-deps

For Mac/Linux:
rm -rf node_modules
npm install --legacy-peer-deps

================================================

COMMAND 2 - Run the development server:

npm run dev

================================================

http://localhost:3000
